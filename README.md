# Vseti для Samsung Tizen

Готовое приложение **Vseti** для телевизоров Samsung на Tizen.

## Скачать

Откройте раздел **Releases** и скачайте последний файл `Vseti_Tizen_v*.wgt`.

## Установка

Для установки стороннего WGT на телевизоре должен быть включён Developer Mode. Установка выполняется через Tizen Studio / Device Manager с сертификатом, разрешающим DUID вашего телевизора.

Исходники и внутренние настройки сборки в этом публичном репозитории не публикуются.
